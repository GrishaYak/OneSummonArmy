# One Summon Army
This is a Terraria mod that adds alternative versions of vanila summoner's staffs. New staffs will only summon one minion and then stregthen it, depending on how much minion slots you've spent on it.

Installation
--
You can install it from Steam's Workshop directly in your tModLoader.

More information
--
This mod in not ready yet, and we're not really working on it lately to be honest. There are 4 staffs ready: Bird staff, Flinx staff, Hornet staff and Slime staff. But there are some annoying bugs, so mod is not really playable for now.
